## 💡 Proposta de Valor

O ReCiclo é um aplicativo que conecta pessoas que desejam doar itens com outras que precisam, promovendo sustentabilidade, reaproveitamento de materiais e redução de desperdício.

---

## 📱 Descrição das Telas

### 🏠 Tela Home / Feed

Exibe os itens disponíveis próximos ao usuário. Possui barra de busca, botão de filtro e lista de itens com imagem, nome e distância.

---

### 🔎 Tela de Filtros

Permite ao usuário refinar sua busca por:

* Distância
* Categoria (roupas, móveis, eletrônicos)
* Estado de conservação

---

### 📦 Tela de Detalhes do Item

Mostra informações completas do item selecionado:

* Imagem
* Nome
* Descrição
* Nome do doador
* Botão para demonstrar interesse

---

### ➕ Tela de Cadastro de Item

Permite que o usuário cadastre um item para doação, informando:

* Foto
* Nome
* Descrição
* Categoria

---

### 💬 Tela de Chat

Tela de conversa entre doador e interessado para combinar a retirada do item.

---

## 🔄 Jornada do Usuário

O usuário inicia o aplicativo na tela inicial (Home), onde visualiza itens disponíveis.
Ele pode aplicar filtros para refinar a busca ou selecionar um item para ver seus detalhes.

Ao se interessar, o usuário inicia uma conversa pelo chat para combinar a retirada.
Caso queira doar, o usuário pode acessar a tela de cadastro e publicar um novo item.

---

## 🧠 Observações de Design

* O protótipo foi desenvolvido em baixa fidelidade (wireframe)
* Utiliza apenas tons de cinza
* Foco na estrutura, navegação e organização da informação
* Elementos representados por formas simples (caixas e linhas)
